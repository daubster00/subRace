# 요구사항 자료 보관소

이 폴더는 SubRace(라이브서브랭크) 프로젝트의 요구사항 원본 자료와 정리본을 보관합니다.

## 구성
- `brief.md` — 고객 자료를 정리한 요구사항 브리프 (메인 에이전트의 입력)
- 원본 고객 자료는 프로젝트 루트의 `자료/` 폴더에 있습니다.

## 원본 자료 위치
프로젝트 루트의 `자료/` 폴더에 다음 자료가 있습니다.

| 파일 | 종류 |
|------|------|
| `자료/프로젝트 브리프 문서.docx` | 개발 방향 정리서 (기능/UX 요구) |
| `자료/유튜브_구독자_라이브_순위_작업_안내서.docx` | 작업 진행 안내서 (스택/일정/계정 준비) |
| `자료/작업_안내서.pdf` | 작업 안내서 PDF 버전 |
| `자료/SubRace_design_100.png` | 최종 디자인 시안 (TOP 100 기준) |
| `자료/SubRace_design_50.png` | 최종 디자인 시안 (TOP 50 기준) |
| `자료/6795a0f8-7393-44fd-a13d-a8e2e61fe637.webp` | 추가 디자인 이미지 |
| `자료/live_subrank_coding/` | 작동하는 프로토타입 (HTML/CSS/JS) |
| `자료/API키.txt` | API 키 (커밋 금지, .gitignore 처리 필요) |
